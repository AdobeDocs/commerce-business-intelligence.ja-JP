# データユーザー

データを使用すると、ビジネス上の意思決定に役立ちます。 チームのデータアナリストからレポートやダッシュボードを受け取る場合もありますが、これらのレポートやダッシュボードの作成方法の詳細も必要になります。

この節では、次の方法を説明します。
* データの取得と表示： [!DNL Commerce Intelligence] レポートとダッシュボードの使用
* メールを通じてそのデータを関係者と共有する `.csv` ファイル
