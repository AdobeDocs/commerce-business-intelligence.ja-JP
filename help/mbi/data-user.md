# データユーザ

データは、ビジネス上の意思決定に役立ちます。 チームのデータアナリストからレポートやダッシュボードを受け取る一方で、これらのレポートやダッシュボードの作成方法を学ぶこともできます。

この節では、以下の方法について説明します。
* データを取得して表示する [!DNL Commerce Intelligence] レポートとダッシュボードの使用
* 電子メールを通じてそのデータを関係者と共有し、 `.csv` ファイル
