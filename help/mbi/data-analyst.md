# Data Analyst

クエリを設計し、データ分析に積極的に参加できます。 データに関する質問に対する具体的な回答を見つけ出し、同僚へのセルフサービス体験を促進する方法を身に付けています。

この節では、次の方法を示します。
* データを接続して[!DNL Commerce Intelligence]にインポート
* Data Warehouseでのデータの保存と構造化を確認できます
* データの分析
* [!DNL SQL Report Builder]を使用
