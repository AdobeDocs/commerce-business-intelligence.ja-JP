# 請求情報

アカウントの請求情報を表示、追加、変更するには、次のものが必要です `Billing` 権限。 [権限はユーザーごとに設定されます](../../administrator/user-management/user-management.md) ユーザーアカウントの作成時。 必要な場合 `Billing` 権限については、アカウントの管理者にお問い合わせください。
