# 請求情報

アカウントの請求情報を表示、追加、変更するには、 `Billing` 権限。 [権限はユーザーごとに設定されます](../../administrator/user-management/user-management.md) ユーザーアカウントの作成時。 必要に応じて `Billing` 権限については、アカウントの管理者にお問い合わせください。
