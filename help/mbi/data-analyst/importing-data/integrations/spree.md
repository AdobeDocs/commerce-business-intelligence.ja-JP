# 接続 [!DNL Spree]

ビジネスのパフォーマンスを確認したり、最も価値のある獲得ソースやチャネルを見つけたりしたい場合は、 接続 [!DNL Spree] データを [!DNL MBI] では、これらの質問などに答えることができます。

この [!DNL Spree] データベースの基準 `PostgreSQL`を使用する場合、 [!DNL MBI] の使用 [通常の手順と同じ `Postgres` 接続](../integrations/postgresql.md).
