# 予測

より多くの注文を行う際に、顧客の生涯価値を予測できることは、あらゆる規模のビジネスにおいて最も重要な側面の 1 つです。

このセクションでは、現在の顧客のライフタイム値について説明し、注文が増えるにつれてライフタイム値がどのように増加するかを予測します。

[ 期待生涯価値（LTV）分析（基本） ](../../data-analyst/analysis/ess-expected-ltv.md) も参照。
