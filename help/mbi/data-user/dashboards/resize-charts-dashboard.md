# ダッシュボードでのチャートの操作

スカラー数値： 棒グラフ： 長期間にわたって拡張されるチャート： グラフごとに情報の表示が異なるため、グラフのサイズと場所は、画一的なソリューションではありません。 [!DNL Commerce Intelligence]では、グラフのサイズを変更したり並べ替えたりして、理想的なワークスペースを作成できます。

*グラフのサイズを変更するには、*&#x200B;をクリックして、任意のグラフの右下隅をドラッグします。

![ グラフのサイズ変更](../../assets/Resize_Chart_in_Dashboard.gif)

*グラフを移動するには、* カーソルが表示されるまで、グラフの上部にカーソルを合わせます`Move`。 クリック&amp;ホールドしてから、チャートを目的の場所にドラッグします。 クリックしてチャートを配置します。

![ グラフを移動](../../assets/Move_Chart_in_Dashboard.gif)
