# ダッシュボードを残す（共有しない）

チームを変更できますか？ [!DNL Commerce Intelligence] アカウントのクリーニングを開始しますか？ ダッシュボードから退出または共有解除するには、退出するダッシュボードで画面上部の&#x200B;**[!UICONTROL Shared]**&#x200B;をクリックします。 **[!UICONTROL Leave this Dashboard]**&#x200B;をクリックして自分自身を削除します。
