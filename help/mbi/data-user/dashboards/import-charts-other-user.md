# 別のユーザーからチャートをインポート

別のユーザーが所有するチャートを編集したり、類似したものを作成したりすることもできます。 現在、別のユーザーが所有し、独自のダッシュボードに保存されているグラフを簡単にインポートできます。

## グラフを検索

まず、他のユーザーからコピーするグラフを見つけます。 共有されているすべてのダッシュボードは、`Dashboard` サイドバーに表示され、共有アイコンでマークされます。 目的のダッシュボードをクリックします。

## グラフの複製

共有ダッシュボードで、自分のアカウントにコピーするグラフを選択します。 歯車（![歯車アイコン &#x200B;](../../assets/gear-icon.png)）アイコンをクリックし、**[!UICONTROL Save As]**&#x200B;をクリックします。

グラフのコピーに名前を付け、既存のダッシュボードに追加するように求められます。 ダッシュボードを選択しない場合、ダッシュボードは既存のグラフのリストに追加され、[後で追加できます](../../data-user/dashboards/add-charts-dashboard.md)。
