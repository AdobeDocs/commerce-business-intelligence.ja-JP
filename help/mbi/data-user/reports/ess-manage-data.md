# データを管理

「データを管理」を使用すると、統合、レポートおよびグラフのデータ、ダッシュボード、エクスポートを管理するための様々なツールにアクセスできます。

## アクセスするには `Manage Data`:

1. メニューで、 **[!DNL Manage Data]**.

1. サイドバーで、次の見出しの下で必要なトピックを選択します。

   * `Data Sources`
   * `Manage Data`
   * `Export Data`
   ![データを管理](../../assets/magento-bi-manage-data.png)<!--{: .zoom}-->
