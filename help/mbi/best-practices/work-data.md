# データの操作

MBI を使用すると、複数のソースからデータをインポートできます。 この節では、そのデータを整理および使用する際のベストプラクティスについて説明します。

データベースと SQL クエリを最適化する方法を学びます。更新サイクルの状態を確認する方法、および更新サイクルの時間を短縮する方法。
